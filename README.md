### 전체 개발 순서

#### Phase 1 — 뼈대 (데이터 & 상태)

```
1. 파일 구조 생성 (index.html, style.css, js/ 폴더들)
2. 디지몬 데이터 작성 (digimon.js)
3. 진화 트리 데이터 작성 (evolution.js)
4. 게임 상태 객체 설계 (game.js)
```

#### Phase 2 — 핵심 로직

```
5. 시간 시스템 (게임 내 시간 흐름, 나이 증가)
6. 육성 시스템 (먹이주기, 훈련, 수면, 케어 실수 카운트)
7. 진화 시스템 (조건 체크 → 진화 실행)
```

#### Phase 3 — 배틀

```
8. 배틀 로직 (자동 전투, 스탯 계산)
9. 토너먼트 시스템 (타이틀 매치, 챔피언십)
10. 테이머 랭크 시스템
```

#### Phase 4 — UI

```
11. 기본 화면 렌더링 (육성 화면)
12. 진화 화면
13. 배틀 화면
14. 상점 / 헌팅 화면
```

#### Phase 5 — 마무리

```
15. localStorage 저장/불러오기
16. 디지몬 200마리 데이터 추가
17. 밸런싱 & 버그 수정
```
